从权威源找到三篇不同视角的研报

系统识别了白银领域的权威信息源，搜索到三篇覆盖不同范式的研报：国泰君安（期货展望，最详细的 45 页）、方正证券（供需结构 + 历史复盘，23 页）、银河证券（宏观周报视角，13 页）。

三篇加起来 81 页。如果你自己读，大概需要一整天。

金字塔原则拆解每篇研报的逻辑框架

严格遵循 Minto 金字塔原理，对每篇研报进行深度逻辑拆解。每个标题是方向性结论而非分类标签，用 IB Color Code 标注数据可信度（蓝=引用、绿=假设、黑=计算），补充逻辑跳步让非专业读者也能看懂。

你可以在原文复刻和逻辑重组两种视图间切换，像在读一份结构化的投研备忘录。

多 Agent 并发，既审数据来源，也审逻辑

每个数据节点和每条逻辑推导各分配一个独立的验证 Agent，全部并发执行。每个 Agent 独立搜索一手来源（Silver Institute、CME、LBMA），比对数据准确性，检验因果链条是否成立。

沉淀核心判断与可追踪的知识资产

从验证结果中筛选出两类资产：

核心判断：经过验证的、对投资主线有决定性贡献的关键事实。红色剔除，黄色择优，绿色中只保留对故事有贡献的。

知识资产库：该领域研究需要的核心 Source，每个都有可点击的一手来源 URL。这不是一次性产物，而是未来持续追踪和深入研究的起点。

三条逻辑线，合并为一条投资主线

结合金字塔原理，把三篇研报提炼出的逻辑框架合成新的主线。这条主线符合 MECE 原则，按 WHY（为什么有结构性问题）、HOW（问题怎么变成危机）、CONTEXT（宏观大背景）、RISK（什么情况会反转）组织。

像是你自己花一整天读完 81 页后，在脑子里形成的那条主线。

从主线导出定制化追踪看板

根据投资主线确定需要追踪的核心指标，按供需面（供需平衡、缺口趋势、ETF 持仓）、现货面（COMEX 库存、LBMA 库存、Lease Rate）、宏观面（FedWatch、金银比、DXY）三大框架组织。

每个指标附 YTD 趋势分析和近期异动预警。追踪过程中复用"沉"步沉淀的 Source 和逻辑框架，让你持续判断主线还成不成立。
