核心原则
投行级 Color Code
用投行的颜色标注体系标识数据可信度：蓝色=引用、绿色=假设、黑色=计算。一眼区分事实与判断。
咨询业方法论
MECE 原则（相互独立、完全穷尽）+ Storytelling 方法，讲出一条优于市面水平的逻辑主线。
社会科学验证范式
三色染色（绿=验证通过、黄=待验证、红=有冲突），独立验证每个数据节点与枝干间的逻辑关系，把主观表述转化为可证伪的严谨推导。
