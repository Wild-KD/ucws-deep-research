Design Principles
4-Color Verification 四色验证
Every data point earns a trust signal from independent source verification.
Verified Unverified Conflict Structural
Anti-Hallucination 反幻觉
Decompose step uses only data in the original report. No self-calculation, no unit conversion, no cross-source splicing. Every data point carries a page number.
Multi-Agent Fan-out 多Agent扇出
Step 3 spawns one sub-agent per data node and causal edge: 19 concurrent verification agents for a 45-page report.
Verification Time-point 验证时点
Data is verified against what was true at report publication date, not today. "Expectations changed" is an update, not an error.
Storyteller Merging 叙事合并
Before merging, determine the story: WHY / HOW / CONTEXT / RISK. "Does it contribute to the story?" outranks "Is it correct?"
Skills as Prompts 技能即提示
Reasoning logic lives in readable Markdown Skill files, not buried in Python code. Judges can read the agent's thinking process directly.
