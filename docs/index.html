投行规范

Financial Model Color Code

借鉴投行财务模型的颜色标注体系：

  • 蓝色 = 引用外部数据源
  • 绿色 = 假设与预测
  • 黑色 = 计算结果

一眼区分哪些是事实、哪些是判断。

咨询方法论

MECE + Storytelling

像顶级咨询顾问一样组织信息：

  • MECE 原则：相互独立、完全穷尽，确保不遗漏、不重复
  • Storytelling：把零散的数据点串成一个有说服力的投资故事
  • 金字塔原则：结论先行，每个标题都是方向性判断

不是罗列事实，而是讲出一条逻辑主线。

社科验证

交叉验证 · 反模型幻觉

用社会科学的研究范式验证 AI 产出：

  • 数据节点验证：每个数据独立溯源到一手来源
  • 逻辑推导验证：每条因果关系独立检验
  • 多 Agent 交叉验证：多个 Agent 各自独立验证，交叉比对

把"听起来对但经不起推敲"的幻觉筛出来。
