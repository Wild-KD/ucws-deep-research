Design principles

4-color verification
Every data point earns a trust signal. Verified Unverified Conflict Structural
Anti-hallucination
Decompose step uses only data explicitly in the report. No self-calculation, no unit conversion, no cross-source splicing. Every data point carries a page number.
Multi-agent fan-out
Step 3 spawns one sub-agent per data node and causal edge: 19 concurrent verification agents for a 45-page report, each searching independently.
Verification time-point
Data verified against what was true at report publication date, not today. "Expectations changed" is an update, not an error.
Storyteller merging
Before merging, determine the story: WHY, HOW, CONTEXT, RISK. "Does it contribute to the story?" outranks "Is it correct?"
