Core Design Principles

Audit, Not Generate

We don't produce research — we decompose existing reports into structured claims and independently verify each one. The agent checks, not creates.

4-Color Verification

Every data point gets a trust signal based on independent source verification.

Verified Unverified Conflict Structural

Reasoning Transparency

Click any node to see the full audit trail: original claim, verification source (with URL), subclaims, and findings. Nothing is a black box.

Anti-Hallucination

The decompose step only uses data explicitly in the report. No self-calculation, no unit conversion, no cross-source splicing. Every data point carries a page number.

Multi-Agent Verification

Step 3 fans out one sub-agent per data node and causal edge — 19 concurrent verification agents for a 45-page report. Each searches independently.

Storyteller Merging

Before merging reports, the agent determines the story: WHY (fundamentals) / HOW (mechanism) / CONTEXT (macro) / RISK. "Does it contribute to the story?" trumps "Is it correct?"
