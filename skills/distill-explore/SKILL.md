# 沉·探：Source 注册表 → 数据地图

你收到"沉·整"技能产出的 Source 注册表。对其中每个一手数据源进行深挖，mapping 它的完整数据版图，发现研报中没有提到但对投资研究有价值的数据。

---

## 本技能不做的事

- 不验证数据准确性（那是"审"技能）
- 不构建注册表（那是"沉·整"技能）
- 不判断投资主线（那是"合"技能）
- 不下载付费数据（只 mapping 免费可获取的部分）

---

## 第 1 步：筛选深挖目标

从 Source 注册表中，筛选值得深挖的 Source：

| 条件 | → 是否深挖 |
|------|-----------|
| 一手数据源 + URL 可用 + 权威/可靠 | ✅ 必须深挖 |
| 官方统计 + URL 可用 | ✅ 值得深挖 |
| 二手引用 | ❌ 跳过（去找它引用的一手源） |
| URL 不可用或付费墙 | ❌ 跳过（标记为"需替代方案"） |

---

## 第 2 步：Mapping 每个 Source

对每个深挖目标，执行以下探索：

### 2.1 访问主页

用 `web_fetch` 打开 Source 的主 URL。记录：
- 这个机构/网站是做什么的？
- 它的数据产品有哪些？（菜单、导航、子页面）

### 2.2 探索数据目录

用 `web_search` 搜索 `site:source-domain.com data` 或 `site:source-domain.com statistics`，发现该 Source 提供的所有数据维度。

### 2.3 逐个数据维度登记

对发现的每个数据维度，记录：

```
维度名称: COMEX白银库存
数据类型: 时间序列
更新频率: 日度
时间范围: 2000-至今
获取方式: 免费网页 / API / 需注册 / 付费
URL: https://...
格式: 网页表格 / CSV下载 / JSON API
与投资主线的关系: 直接相关（现货面核心指标）
```

### 2.4 判断与投资主线的关系

对每个发现的数据维度，问：

→ 这个数据和我们的投资主线有关系吗？
  → **直接相关**：是投资主线中某个论点的核心数据（如供需缺口）
  → **间接相关**：能辅助判断但不是核心（如白银在某个应用领域的消费量）
  → **潜在价值**：目前主线没用到，但如果主线修正可能会用到
  → **无关**：和白银投资完全无关，跳过

---

## 第 3 步：发现研报遗漏的数据

对比研报实际引用的数据 vs Source 完整数据版图，找出：

| 类型 | 举例 |
|------|------|
| **研报引用了但没充分利用的** | 报告引用了WSS的年度数据，但WSS其实还有季度数据 |
| **研报完全没提到的** | Silver Institute有月度ETF持仓追踪，但报告没用 |
| **同一Source的不同粒度** | 报告用了年度数据，但月度/日度数据也存在 |
| **同一Source的历史回溯** | 报告只看了近两年，但数据可以回溯10年 |

---

## 第 4 步：输出数据地图

### 4.1 每个 Source 一张地图

```
┌─────────────────────────────────────────┐
│ Silver Institute                         │
│ silverinstitute.org                      │
│                                         │
│ ┌─ 已用 ─────────────────────────┐      │
│ │ · 年度供需平衡 (年度, 2015-26E) │      │
│ │ · 矿产银产量 (年度)             │      │
│ │ · 光伏用银 (年度)               │      │
│ └────────────────────────────────┘      │
│                                         │
│ ┌─ 未用但直接相关 ──────────────┐       │
│ │ · 月度ETF持仓 (月度, 免费)     │       │
│ │ · 银价历史 (日度, 免费)        │       │
│ └────────────────────────────────┘      │
│                                         │
│ ┌─ 潜在价值 ──────────────────┐         │
│ │ · 再生银来源分析 (年度, 付费)  │         │
│ │ · 区域消费分布 (年度, 付费)    │         │
│ └────────────────────────────────┘      │
└─────────────────────────────────────────┘
```

### 4.2 推荐纳入追踪的数据

从"未用但直接相关"中，推荐哪些应该纳入"追"步的看板追踪：

→ 更新频率够高（至少月度）？
→ 免费可获取？
→ 对判断投资主线是否仍然成立有帮助？

全部满足 → 推荐纳入追踪
