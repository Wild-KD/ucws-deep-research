# 搜索：给定标的 → 找到 3+ 篇研报

你收到一个投资标的或研究主题。搜索至少 3 篇来自不同范式的研报或深度文章。

---

## 本技能不做的事

- 不拆解报告（那是"读"技能）
- 不验证内容（那是"审"技能）
- 不评价报告质量——只负责找到并返回元数据

---

## 第 1 步：确定搜索范式

至少覆盖 3 种不同视角。根据标的类型选择：

| 标的类型 | 推荐范式组合 |
|---------|------------|
| 大宗商品 | 期货公司研报 + 券商宏观周报 + 行业协会/媒体 |
| 个股 | 券商深度 + 买方观点/基金 + 财经媒体 |
| 行业/赛道 | 行业研究院 + 咨询公司 + 独立分析师/博主 |
| 宏观 | 央行报告 + 券商宏观 + 国际机构(IMF/BIS) |

→ 目标不是找 3 篇"差不多"的报告，而是找 3 篇**视角不同**的报告。如果都来自同一类机构，再搜一轮。

---

## 第 2 步：执行搜索

### 2.1 关键词策略

构造至少 3 组搜索词：
1. `{标的} 研报 {年份}` / `{标的} outlook {year}`
2. `{标的} 供需 分析` / `{标的} supply demand`
3. `{标的} 深度报告` / `{标的} research report`

### 2.2 搜索执行

使用 `web_search` 依次搜索。每组取前 5 条结果。

### 2.3 去重

→ 同一机构的多篇报告只保留最新/最深的一篇
→ 转载/二手解读不算，只要原文

---

## 第 3 步：筛选

对每篇候选报告，逐条检查：

| 检查项 | 不通过则跳过 |
|--------|------------|
| 有完整论证链（不只是结论）？ | 跳过纯观点/快讯 |
| 有数据支撑（表格/图表/数字）？ | 跳过纯定性评论 |
| 页数 ≥ 5 页？ | 跳过太短的 |
| 来源可信（机构/知名作者）？ | 跳过匿名自媒体 |
| 发布时间在最近 12 个月内？ | 跳过太旧的（除非是经典） |

---

## 第 4 步：提取元数据

对通过筛选的每篇报告，提取：

```json
{
  "id": "简短标识符",
  "title": "完整标题",
  "source_org": "机构名称",
  "author": "作者（如有）",
  "publish_date": "YYYY-MM-DD",
  "pages": 0,
  "url": "下载或阅读链接",
  "core_view": "一句话核心观点",
  "paradigm": "卖方研报 | 买方观点 | 媒体深度 | 行业报告 | 独立分析"
}
```

---

## 第 5 步：检查多样性

审视最终列表：
→ 是否覆盖了 3 种以上范式？如果都是券商研报，再搜一轮找买方或媒体。
→ 是否有中文和英文？如果标的是国际性的，最好中英都有。
→ 页数和深度是否有层次？理想是一篇长深度 + 一篇中等 + 一篇短观点。

---

## 输出格式

```json
{
  "topic": "标的名称",
  "search_date": "YYYY-MM-DD",
  "reports": [
    {
      "id": "string",
      "title": "string",
      "source_org": "string",
      "author": "string",
      "publish_date": "YYYY-MM-DD",
      "pages": 0,
      "url": "string",
      "file_path": null,
      "core_view": "string",
      "paradigm": "string"
    }
  ]
}
```
